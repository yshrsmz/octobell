# ユーザーへの質問は AskUserQuestion を使う

ユーザーに判断・選択・確認を求めるときは、原則として `AskUserQuestion` ツールを使うこと。地の文で選択肢を並べて「どれにする？」と聞くのではなく、構造化された選択肢として提示する。

**適用する場面**
- 方針・経路・粒度など、複数案から選んでもらうとき
- 「このまま進めていい？」の最終確認
- 前提が曖昧で、ユーザーの意図を確認しないと進めないとき

**例外（地の文でよい）**
- コードや既定値、文脈から自明に決まること（その場合は決めて進め、何を選んだか一言添える）
- 純粋に情報を伝えるだけで、ユーザーの回答が次の行動を変えないとき
